# Summer 2026 Memory Book

A personal family project built around a 3-week summer holiday in Póvoa de
Varzim, Portugal. We walk the town, beach and city park completing small
location-based tasks and photographing each one, turning the result into a
keepsake memory book (a physical photo-map + per-task pages) plus a shareable
digital twin — and along the way, a hands-on way to learn real geolocation /
geodata skills.

## Folder layout

- `src/` — application code
- `photos/` — local test images (contents gitignored, never committed)
- `output/` — generated maps/books (contents gitignored, never committed)
